# SillyTavernCardCreator

一个轻量的单页应用（SPA），用于编辑 SillyTavern 角色卡配置，并支持导入 / 复制 / 下载 Markdown。

## 使用方式

1. 直接打开 `index.html`（或通过静态服务器访问）。
2. 所有字段默认留空，卡片上显示示例占位内容；点击任意字段卡片可放大编辑。
3. 点击「粘贴导入」可从剪贴板一键解析 `## 字段名 + 内容` 格式 Markdown。
4. 点击「复制 Markdown」可将当前配置复制到剪贴板。
5. 点击「下载 Markdown」可一键下载 `sillytavern-card.md`。
